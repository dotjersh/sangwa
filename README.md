# sangwa
JS and PHP text-encoder

### JS Usage
`sangwa.encode(value);`
returns encoded text

`sangwa.decode(encodedText);`
returns regular text

### PHP Usage
`sangwa_encode($value);` returns encoded text

`sangwa_decode($encodedText);` returns regular text
